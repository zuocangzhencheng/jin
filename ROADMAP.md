# 路线图

## 近期优先级

### 1. 数据可靠性增强

- 增加备用行情源。
- 标注每个模块的数据更新时间。
- 区分实时行情、延迟行情、失败行情。
- 当免费源失败时提供更明确的降级提示。

### 2. 复盘能力增强

- 按日期筛选复盘日志。
- 按股票筛选复盘日志。
- 自动生成“今日最值得复盘的 5 只票”。
- 增加早盘和晚间总结导出格式。

### 3. 买卖点进一步严格化

- 增加“突破后回踩不破”确认。
- 增加“放量长上影”风险提示。
- 增加“连续缩量阴跌”提示。
- 增加“板块强而个股弱”的剔除逻辑。

### 4. 主线识别升级

- 主线强度从单日扩展到多日。
- 识别板块连续性，而不是只看当天涨跌。
- 记录主线第一次异动、二次确认、高潮、退潮。

### 5. 云端与通知

- 研究部署到 GitHub Pages、Vercel 或个人服务器。
- 增加桌面通知。
- 增加声音提醒。
- 研究可选的付费行情接口。

## 中期方向

- 增加可视化 K 线图。
- 增加板块轮动图。
- 增加交易计划模板。
- 增加仓位管理建议。
- 增加统计回测：哪些信号更容易成功，哪些信号是假动作。

## 长期方向

- 做成真正的个人投研工作台。
- 支持多设备同步。
- 支持云端定时任务。
- 支持自定义策略规则。
- 支持更可靠的专业行情数据。
