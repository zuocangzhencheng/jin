# 更新日志

## v1.3 交易复盘

- 新增每只自选股的“为什么盯 / 什么条件买 / 错了在哪里卖”三问。
- 新增“自选同步 / 复盘日志”模块。
- 支持复制自选同步文本。
- 支持下载和导入 JSON 备份。
- 支持手动复盘。
- 自动记录严格买点、卖点、强观察和数据异常。

## v1.2 智能新增

- 新增“代码或名称”输入。
- 输入 6 位股票代码可自动补名称。
- 输入常见股票名称可自动补代码、主线和逻辑。
- 对科创板、创业板、北交所给出权限和波动提醒。

## v1.1 黄金监控

- 新增黄金价格模块。
- 金价随主刷新每分钟更新。
- 增加黄金与风险偏好观察维度。

## v1.0 主线盯盘

- 自选股卡片化展示。
- 市场环境、主线强度、资金异动雷达、主线聚合雷达。
- 集合竞价观察。
- K 线状态、买点缺口、数据可信判断。
- 严格买卖点和风控提示。
